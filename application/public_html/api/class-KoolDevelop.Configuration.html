Class Configuration

Configuration

Reads and parsers configuration files. These configuration files are put in the CONFIG_PATH of your application. The .ini files are parsed with PHP so you can use PHP for special needs.

It is possible to use specific configuration files based on the current environment place the specific configuration files in a subfolder and call \\KoolDevelop\\Configuration::setCurrentEnvironment()
